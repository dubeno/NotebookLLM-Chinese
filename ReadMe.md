# 直接使用
请查看右侧TAG，可以下载中文版本的zip包

# 如何开发运行

npm install 安装依赖

本地运行: 
    npm start
导出windows exe: 
    npm run build
    
程序设置

需要设置APIkey 和角色配置，目前使用本人共享密钥，额度用完就无法使用，需要自己申请密钥
